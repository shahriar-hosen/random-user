# Random User React App

A simple React application that fetches and displays random user profiles using the Random User API. This project was built to practice React fundamentals such as components, props, state management, side effects, and API integration.

## 🚀 Features

* Fetches random users from the Random User API
* Displays user profile information
* Responsive user cards
* Built with reusable React components
* Uses React Hooks (`useState` and `useEffect`)

## 🛠️ Built With

* React
* JavaScript (ES6+)
* CSS
* Random User API

## 📦 Installation

1. Clone the repository:

```bash
git clone https://github.com/shahriar-hosen/random-user.git
```

2. Navigate to the project folder:

```bash
cd random-user
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm start
```

The application will run at:

```
http://localhost:3000
```

## 📡 API Used

This project uses the free Random User API:

https://randomuser.me/
